== Installation ==

1. 	Install & Activate NextGEN Gallery

2.	Add a NextGEN Gallery and upload some images

3. 	Install & Activate Slidepress

4. 	Install & Activate this plugin

5.	Add a Slidepress Gallery

6.      Select XML source "default" and add f.e. /wp-content/plugins/nggslidepressxml/xml.php?mode=gallery&gid=23

That's it ... Have fun


== Frequently Asked Questions ==

Which modes are supported?

For a single gallery /wp-content/plugins/nggslidepressxml/xml.php?mode=gallery&gid=23
For an album /wp-content/plugins/nggslidepressxml/xml.php?mode=album&aid=5

Can I add url tags?

Yes, just add &url=1 to your rss request url, f.e. /wp-content/plugins/nggslidepressxml/xml.php?mode=gallery&gid=23&url=1
